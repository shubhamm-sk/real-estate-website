# Auctions & More — Real Estate Auction Website

A premium, fully responsive real estate auction website built with
pure HTML, CSS, and vanilla JavaScript. No build step, no dependencies —
just open `index.html` in a browser.

## ✨ Features
- Sticky responsive navbar with mobile hamburger menu
- Hero section with live countdown timer and animated stats
- Property categories (Residential, Commercial, Land, Retail)
- Live auction cards with bid info and status tags
- "How It Works" 4-step process
- Why-choose-us trust section
- Working contact form with success feedback
- Scroll-reveal animations
- Fully responsive (desktop, tablet, mobile)
